# Сверка документов

Локальный веб-сервис для автоматического сопоставления двух таблиц Excel по дате и сумме. Ориентирован на бухгалтеров и финансовых специалистов.

## Что умеет

- Принимает два файла `.xlsx` / `.xls` с любой структурой столбцов
- Позволяет вручную выбрать колонку даты и колонку суммы для каждого файла
- Поддерживает файлы без строки-заголовка
- Четыре режима допуска по дате:
  - Только точные совпадения
  - ±1 календарный день
  - ±1 рабочий день (пятница → понедельник считается как 1 день)
  - ±3 дня
  - ±7 дней
- Групповое совпадение: одна строка = сумма нескольких строк другого файла
- Результат — цветной Excel с разделами «Совпадения» и «Расхождения»
- Файл сохраняется в папку `result_reports/` и скачивается через браузер

## Цветовая легенда результата

| Цвет   | Значение                              |
|--------|---------------------------------------|
| Зелёный | Точное совпадение (дата + сумма)     |
| Жёлтый  | Совпадение с допуском по дате        |
| Оранжевый | Групповое совпадение (1:N или N:1) |
| Красный | Расхождение — строка не найдена      |

## Алгоритм сопоставления

Жадный, четырёхфазный (порядок важен — каждая фаза «забирает» строки до следующей):

1. **Точное совпадение** — одинаковые дата и сумма (погрешность < 0,005)
2. **Группа А→Б** — одна строка А = сумма нескольких строк Б в пределах допуска по дате
3. **Группа Б→А** — одна строка Б = сумма нескольких строк А
4. **Допуск 1:1** — строки с одинаковой суммой и датой в пределах допуска

Группы ищутся раньше допуска, чтобы строки не «уходили» в толерантные пары вместо правильных групп.

## Поддерживаемые форматы дат

- Дата Excel (Timestamp)
- Число `1.07` → 1 июля текущего года
- Строка `"1.07"` → то же самое
- Строка `"01.07.2025"` → стандартный разбор

## Как запустить

```bat
cd reconciliation
start.bat
```

Или вручную:

```bat
pip install -r requirements.txt
python main.py
```

Открыть в браузере: `http://localhost:8001`

> Порт 8001 выбран вручную — порт 8000 занят другим проектом.

## Структура проекта

```
reconciliation/
├── main.py               # FastAPI: эндпоинты /api/preview, /api/reconcile, /api/download
├── reconciler.py         # Логика парсинга, сопоставления и формирования Excel
├── requirements.txt
├── start.bat
├── static/
│   ├── index.html        # Интерфейс (3 шага: загрузка → настройка → результат)
│   ├── style.css
│   └── script.js
└── result_reports/       # Сохранённые результаты сверки (имя: sverka_YYYYMMDD_...)
```

## Технологии

- **Backend:** Python 3.12, FastAPI, pandas, openpyxl
- **Frontend:** Vanilla JS, HTML/CSS (без фреймворков)
- **Сервер:** uvicorn на `127.0.0.1:8001`
